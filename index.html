Securus

Securus is an effective, minimal and highly robust user login and register system based entirely on PHP and MySQLi (MySQL Improved). Use this and be rest assured that your site is safe from most common forms of hacking attacks.

Download as .zip Download as .tar.gz View on GitHub

Secures your site from:

  • SQL Injections
  • Session Hijacking
  • Network Eavesdropping
  • Cross Site Scripting
  • Brute Force Attacks

About

A lot of people know how to create PHP and mySQL based websites but don't know how to protect their sensitive data from most common forms of hacking attempts. I searched for a lot of solutions across the web and compiled the best ones into one unified secured system that you can use as a sandbox before building a site, to protect your valuable data.
